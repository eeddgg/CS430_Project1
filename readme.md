#Usage
To convert a P6 or P3 PPM image to a P3 image:
    `ppmrw 3 "path/to/input.ppm" "path/to/output.ppm"`
To convert a P3 or P6 PPM image to a P6 image:
    `ppmrw 3 "path/to/input.ppm" "path/to/output.ppm"`
